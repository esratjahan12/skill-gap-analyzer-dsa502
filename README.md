# Skill Gap Analyzer

DSA 502 Phase 1 Project

This project analyzes job postings to identify skill gaps for Data Analyst and Data Scientist roles.

## Dataset
Kaggle – Data Science Job Postings and Skills  
https://www.kaggle.com/datasets/asaniczka/data-science-job-postings-and-skills

## Phase 1
- Data audit
- Exploratory data analysis
- Baseline NLP model

## Phase 2
A Flask application will allow users to input skills and receive:
- Skill Match Score
- Missing Skills
- AI Recommendations
